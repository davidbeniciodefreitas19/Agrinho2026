Objetivo

O site tem como objetivo de mostrar como seu alimento chega na sua mesa. Mostrando cada passo para que o alimento chegue com todos os cuidados possiveis.

Justificativa 

O site foi desenvolvido para fins escolares, para que os usuarios aprendam como o alimento chega na mesa de cada cidadão.

Instruções 

No site vai aparecer uma caixinha para você colocar seu nome e logo abaixo vai ter as explicações de como o alimento chega na mesa da população.
